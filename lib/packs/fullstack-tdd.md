# Fullstack TDD Pack

> Everything you need for professional, test-driven full-stack development. Install this pack and you have the complete feature-development workflow with all its agents and skills.

## What's Included

### Workflow
- **feature-development** — The primary 13-phase, 6-gate development workflow

### Agents (12)
| Category | Agents |
|----------|--------|
| Core | `implementer`, `test-writer`, `architect`, `planner`, `spec-writer` |
| Control | `reviewer-quality`, `consistency-guardian` |
| Delivery | `router`, `onboarder`, `docs-writer`, `changelog-writer`, `performance-monitor` |

### Skills (15)
| Level | Skills |
|-------|--------|
| Core | `debugging`, `refactoring`, `task-management`, `api-design`, `project-memory`, `context-management` |
| Governance | `observability`, `conventional-commits`, `trace-writing`, `preview-delivery`, `consistency-check`, `performance-profiling` |
| Growth | `worktree-management`, `feature-branch`, `e2e-testing` |

## When to Use

- Building new features in any tech stack
- Projects that follow TDD (Red → Green → Refactor)
- Teams that want full traceability and human oversight
- Any project where quality gates are non-negotiable

## Install

```bash
linxmd install packs/fullstack-tdd
```

This installs all 12 agents, 15 skills, and the feature-development workflow.
